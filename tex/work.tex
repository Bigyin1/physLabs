По углу наклона $ k = \lambda / L = 224.5\pm 1.8 $ найдём базу интерферометра:

\paragraph{Жёлтый светофильтр}



Для жёлтого дублета спектра ртути замерили координаты 5 колец:


\input{tex/tables/rtyellow.tex}


По этим данным:

\begin{figure}[H]
    \centering
    \includegraphics[scale=0.4]{rtyellow.png}
    \caption{График $ D_1^2 + D_2^2 $ от номера кольца}
\end{figure}

Из графика найдем среднюю длину волны желтого дублета ртути:
\begin{equation*}
    \frac{D_1^2 + D_2^2}{8f^2} = n \cdot \frac{\lambda_\text{ср}}{L}
\end{equation*}

Тогда: $\lambda_\text{ср} = \frac{L \cdot k}{8f^2} = 570 \pm 20 \text{ } nm$


\begin{figure}[H]
    \centering
    \includegraphics[scale=0.5]{rtyellow2.png}
    \caption{График зависимости $ D_1^2 - D_2^2 $ от номера кольца}
\end{figure}

Из графика найдем разность длин волн желтого дублета ртути:
\begin{equation*}
    \frac{D_1^2 - D_2^2}{4f^2} = n \cdot \frac{\delta\lambda}{L}
\end{equation*}

Тогда: $\delta\lambda = \frac{L \cdot k}{4f^2} = 5 \pm 2.5 \text{ } nm$


Оценим максимальный порядок интерференции $m$ для желтой линии ртути:

\[
    m_\text{жёл} = \frac{2L \cos \theta}{\lambda} \approx \frac{2L}{\lambda} = 345
\]

Оценим дисперсионную область:

\[ \Delta \lambda_{жёл} = \frac{\lambda^2}{2 L}= 149 \; \SI{}{\angstrom}. \]

Найдём разрешающую способность прибора:

\[ R = \frac{\lambda}{\delta \lambda} \approx \frac{1}{\theta_1 \delta \theta_1} = \frac{4f^2}{D \delta r}\]
